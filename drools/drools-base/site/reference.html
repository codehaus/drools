Base Semantic Module Component Reference
Rule

The Rule primitive is actually a port of the core. This semantic module simply adds a default factory for Rule objects.

FixedDuration

Additionally, the core FixedDuration primitive is exposed through the base semantic module. The FixedDuration allows the static definition of the truth duration of a rule. 

FixedDuration duration = new FixedDuration();

duration.addDays( 42 );
duration.addHours( 12 );

rule.setDuration( duration );

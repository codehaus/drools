The Application Data View

The Application Data View shows all application data currently defined in the Drools engine.

Three actions are added to the right of the view, to customize what is shown:

  • The Show Detail Pane action toggles the detail pane at the bottom of the view.
  • The Show Logical Structure toggles showing the logical structure of the application data, or all their details. Logical structures allow for example visualizing sets of elements in a more obvious way.
  • A drop down contains two actions to toggle the display of constants and static variables.
